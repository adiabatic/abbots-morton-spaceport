Add an entry for a letter in @glyph_data/quikscript.yaml. Do not try to determine what the letter should look like. The only thing that should be in the `bitmap` array is a list of strings with only five spaces in it. The number of elements in the `bitmap` array is determined by whether the letter is Tall, Short, or Deep. Tall and Deep letters have 9 items in `bitmap`, while Short letters have 6. Deep letters have a y_offset of -3. Every letter needs a bare trailing `#` comment marker on the two rows whose glyph-space `y` values are 5 (the x-height) and 0 (the baseline); which bitmap rows those are depends on `y_offset`.
