    Keys

    jApprove, then advance to the next unverdicted unit
    fReject, then advance
    dFine either way (any-of), then advance
    kSkip (explicit no-verdict record), then advance
    uUndo the last verdict and return to it
    nFocus the current unit’s note field
    gApprove every remaining unverdicted unit in the current group
    xToggle the current unit’s explain/provenance panel
    ↓ / ↑Move the cursor without verdicting
    [ / ]Previous / next batch
    ?This overlay
    EscapeBlur input / close overlay

    Click a config chip first to scope the next verdict to that config; the default verdict covers all of the unit’s configs.
